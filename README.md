# Android Kotlin - Navegation Between Screens

## Descrição
Projeto para um exercício para a Gs valendo 1 ponto e também para estudo para o Checkpoint 1. Sobre criação de telas(Login, Menu, Pedidos e Perfil), com navegações entre elas, Navegation Compose e uma boa pratica de organização de código de android moderno.
